Sweet spot: Kp = 160 across slopes ≤ 15° and all mass perturbations. Above 15° the controller degrades regardless of Kp — that's not a tuning problem, that's a structural limit of this PD controller (no feedforward for body pitch).

Default is currently Kp = 140. Recommendation: bump to 160, and flag slopes > 15° as out-of-spec in the scenario manifest so the regression harness doesn't grade us on them.
